Frequently asked questions

How do I upgrade to Pro?

Install the extension from the Chrome Web Store, open LinkedIn, then click the extension icon in your toolbar and select Upgrade. Payment is handled securely through Stripe.

Is this a subscription?

No. Pro is a lifetime purchase. You pay once and keep access on every Chrome browser you're signed into — no monthly or annual fees.

Does my license work on multiple devices?

Yes. Your license syncs via Chrome sync, so the same purchase covers every Chrome profile and device where you're signed in.

Can I install without paying?

Yes. Download the extension free from the Chrome Web Store. Upgrade to Pro anytime from the extension popup when you're ready.
